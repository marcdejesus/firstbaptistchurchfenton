Content Width Variations

Narrow Content Layout

This narrow layout (max-width: 800px) is perfect for reading-focused content like sermons, blog posts, or detailed ministry descriptions. The constrained width improves readability and focus.

Standard Content Layout

The standard layout (max-width: 1200px) works well for most pages including ministry overviews, event listings, and general information pages. It provides a good balance between content space and readability.

Wide Content Layout

The wide layout (max-width: 1400px) is ideal for gallery pages, large data displays, or when you need to showcase multiple items side by side. This layout makes full use of larger screens while remaining readable.
